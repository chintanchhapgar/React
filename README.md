# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/chintanchhapgar/React.git
```

Navigate to a project

```bash
cd QuickCart
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npm start
```

or (Vite projects)

```bash
npm run dev
```
